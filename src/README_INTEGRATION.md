```
현재 최적화된 구조:

│   ├── layouts/               # 레이아웃 컴포넌트
│   └── 기타 비즈니스 컴포넌트들
├── styles/globals.css         # 테마 시스템 (다크/라이트)
└── 기타 필수 설정 파일들
```
